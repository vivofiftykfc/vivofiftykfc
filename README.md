## 👋 关于我 / About

我在北京理工大学学习电子信息工程，关注如何把感知算法真正放进系统里：从视觉、红外和毫米波雷达获取现实信号，在嵌入式平台或 FPGA 上完成处理，再把结果组织成能被人使用的 AI 产品。

I enjoy building across the whole path:

<p align="center"><code>Sensing → Edge Computing → Decision → Interaction</code></p>
